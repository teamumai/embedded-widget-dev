Embedded widget QA

All environments use KWC Singapore by default.
To test the widget of another venue, add the API key of that venue as a query parameter to the respective url.

For example to use a different API key on STAGING:
https://teamumai.github.io/embedded-widget-qa/staging?apiKey=ff47ab19-26bb-4eb1-97b3-c0d79ab398c3
